== Partitionierung - Ablauf

[.step]

. **Tausche** Elemente

== Partitionierung - Code

```python
while True:
    # Von links: Element >= Pivot
    while arr[i] < v and i < hi:
        i += 1

    # Von rechts: Element <= Pivot
    while arr[j] > v and j > lo:
        j -= 1

    if i >= j: break
    arr[i], arr[j] = arr[j], arr[i]
```

== Rekursive Sortierung

```python
def quicksort(cls, arr: list, lo: int, hi: int):
    if lo >= hi:
        return arr

    j = cls.partition(arr, lo, hi)

    # Rekursiv sortieren
    cls.quicksort(arr, lo, j - 1)    # Links
    cls.quicksort(arr, j + 1, hi)    # Rechts
```

== Beispiel: [5, 2, 8, 1, 9]

[.step]
* **Pivot**: 5
* Partitioniere: [2, 1] **5** [8, 9]
* Sortiere links: [1, 2]
* Sortiere rechts: [8, 9]
* **Ergebnis**: [1, 2, 5, 8, 9]

== Hauptmethode

```python
@classmethod
def sort(cls, arr: list) -> list:
    if arr is None:
        return None

    arr_copy = arr.copy()
    cls.quicksort(arr_copy, 0, len(arr_copy) - 1)
    return arr_copy
```
