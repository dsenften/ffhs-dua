== Willkommen

Willkommen zur Dokumentation des FFHS-DUA Projekts - einer umfassenden Python-Implementierung grundlegender Algorithmen und Datenstrukturen.

Dieses Projekt wurde für den akademischen Gebrauch an der Fernfachhochschule Schweiz (FFHS) entwickelt und bietet eine vollständige, gut dokumentierte und getestete Sammlung von Algorithmen und Datenstrukturen.

Das `sorting`-Modul enthält verschiedene Sortieralgorithmen:

=== Grundlegende Verwendung

==== Fundamentals

[source,python]
----
from src.algs4.fundamentals import Stack, Queue, Bag
from src.algs4.fundamentals.uf import UF

# Stack verwenden
stack = Stack[int]()
stack.push(1)
stack.push(2)
print(stack.pop())  # 2

# Union-Find verwenden
uf = UF(10)
uf.union(0, 1)
print(uf.connected(0, 1))  # True
----

==== Sorting

[source,python]
----
from src.algs4.sorting import Quick, Shell, Heap, Merge

# Quick Sort
numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = Quick.sort(numbers)
print(sorted_numbers)  # [11, 12, 22, 25, 34, 64, 90]

# Heap Sort (garantierte O(n log n) Performance)
numbers2 = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers2 = Heap.sort(numbers2)
print(sorted_numbers2)  # [11, 12, 22, 25, 34, 64, 90]

# Merge Sort (stabiler Algorithmus)
numbers3 = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers3 = Merge.sort(numbers3)
print(sorted_numbers3)  # [11, 12, 22, 25, 34, 64, 90]

# Shell Sort
words = ["zebra", "apple", "banana", "cherry"]
sorted_words = Shell.sort(words)
print(sorted_words)  # ['apple', 'banana', 'cherry', 'zebra']
----
