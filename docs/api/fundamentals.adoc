== Bag (Sammlung)

Die `Bag`-Klasse implementiert eine ungeordnete Sammlung von Elementen.

=== Klasse: `Bag[T]`

[source,python]
----
from src.algs4.fundamentals.bag import Bag

bag = Bag[int]()
----

==== Methoden

- `add(item: T) -> None`: Fügt ein Element hinzu
- `is_empty() -> bool`: Prüft, ob die Sammlung leer ist
- `size() -> int`: Gibt die Anzahl der Elemente zurück
- `contains(item: T) -> bool`: Prüft, ob ein Element enthalten ist
- `remove(item: T) -> bool`: Entfernt ein Element
- `clear() -> None`: Entfernt alle Elemente

==== Zeitkomplexität

- `add()`: O(1)
- `contains()`: O(n)
- `remove()`: O(n)
- `size()`: O(1)
