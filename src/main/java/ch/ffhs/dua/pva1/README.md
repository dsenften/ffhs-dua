# Aufgabe zu verketteten Listen

Implementieren Sie eine doppelt gekettete Liste sowie eine dazugehörige Iterator-Klasse.
Es dürfen keine Klassen aus dem Package <code>java.util</code> verwendet werden.

Die Listenklasse soll die folgenden Methoden aus dem Interface 
<code>java.util.List</code> implementieren:

- <code>add(E)</code>
- <code>add(int, E)</code>
- <code>get()</code>
- <code>set(int, E)</code>
- <code>remove(int)</code>
- <code>size()</code>
- <code>iterator()</code>

Die Iterator-Klasse sollte möglichst effizient implementiert sein und die 
folgenden drei Methoden  implementieren:

- <code>hasNext()</code>
- <code>next()</code>
- <code>remove()</code>

Verwenden Sie die Klasse <code>LinkedList</code> dieses Projektes und 
ersetzen Sie die TODO's.
